## 🔧 技术细节

### Tailwind CSS 类使用
- `bg-gradient-to-br` - 渐变背景
- `group-hover:` - 群组 hover 效果
- `transition-all` - 所有属性过渡
- `duration-300` - 300ms 动画时长

### React JSX
- 使用 `<a>` 标签作为可点击卡片
- `className` 动态类名
- 保持语义化 HTML
